Your Markdown Files Are
Executable AI Agents

Write prompts in markdown, run them like scripts. Works with Claude, Gemini, Codex, Copilot, and any CLI tool.

Get Started View on GitHub
Terminal
$ review.claude.md # Run with Claude
$ commit.gemini.md "fix auth bug" # Run with Gemini
$ git diff | explain.claude.md # Pipe through any command

Why mdflow?

Stop juggling CLI flags. Start writing agents that are readable, reusable, and composable.

⚡

Filename = Command

Name your file task.claude.md and it runs with Claude. Simple as that.

⚙

YAML = CLI Flags

Every frontmatter key becomes a CLI flag. No translation layer, no magic.

🔗

Unix Philosophy

Pipe data in, chain agents together. Compose AI workflows like shell commands.

📄

Import Anything

Use @./file.ts to inline code, globs for patterns, URLs for remote content.

🎱

Template Variables

LiquidJS templating with {{ _vars }}. Override via CLI flags.

🚀

Zero Config

Works out of the box. Optional global config in ~/.mdflow/config.yaml.

How It Works

Three simple rules. That's all you need to know.

review.claude.md markdown 
---
model: opus
dangerously-skip-permissions: true
add-dir:
  - ./src
  - ./tests
---
Review this code for bugs and suggest improvements.

@./src/**/*.ts
Terminal bash 
# Runs: claude --model opus --dangerously-skip-permissions \
#       --add-dir ./src --add-dir ./tests "Review this code..."

$ review.claude.md

Installation

Get started in seconds.

npm install -g mdflow

Then run mdflow --setup to enable direct execution of .md files.
